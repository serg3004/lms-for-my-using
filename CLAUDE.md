# Правила работы

## Перед каждой задачей
1. `git pull origin main` — получить последние изменения (в том числе от ChatGPT)
2. Создать ветку: `git checkout -b claude/название-задачи`

## После выполнения задачи
1. `git add .`
2. `git commit` — понятное сообщение: что и зачем изменено
3. `git push origin claude/название-задачи`

## Запрещено
- Коммитить напрямую в `main`
- Делать `git push --force`
